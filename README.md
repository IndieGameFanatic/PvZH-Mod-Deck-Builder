A relatively simple WinForms program; This PvZH Modding tool helps you modify both strategy and AI decks much more rapidly!

# Information
On the left side (1), you have the list of all cards in the game and a search bar for finding cards, either by ID or by name.

On the right side (2), you have the deck list, showing you what all the cards in the deck are and how many copies of each.

At the top (3) is additional info about deck, including whether the deck is a strategy or AI deck, on what faction it belongs to, and the name of the deck. If the deck is an AI deck, 
faction and name will be disabled as it is not needed.

If you load in a bundle, a list to the right of the deck list (4) will show up, allowing you to search for decks in the bundle to modify.

# Tutorial: Loading Custom Card Data

If you have a mod with tons of custom cards that you want to use with the program, you can do that very easily! Simply go to 'Card Data > Load Custom Card Data' and select the folder with
both 'cards.txt' and 'localizedstrings.txt' (must be named exactly like that). This will automatically load in every card and get their names, allowing you to easily add custom cards to your decks with no fret! (Syndrome Mod used as a demonstration).

<img width="691" height="576" alt="image" src="https://github.com/user-attachments/assets/5dffec25-109f-49a5-9311-453179ab37bd" />

Additionally, you can also go to the 'autoload' folder and replace both of the files there with your own custom card data. This changes the default card data into the one for your own mod, which should be very convenient if you find yourself constantly modifying decks!
